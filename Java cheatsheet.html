Variables

let
Declares a variable that can be reassigned.
const
Declares a constant variable whose value cannot be changed.
var
Declares a variable with function-scoped or globally-scoped visibility.

Operators

+
Addition operator for adding numbers or concatenating strings.
-
Subtraction operator for subtracting numbers.
*
Multiplication operator for multiplying numbers.
/
Division operator for dividing numbers.
%
Modulus operator for finding the remainder of a division.
==
Equality operator that compares two values for equality.
===
Strict equality operator that compares both value and type.

Functions

function()
Declares a function.
return
Specifies the value returned by a function.
Arrow Function (=>)
A concise way to declare functions (also called a fat arrow).

Conditionals

if/else
Executes a block of code if a specified condition is true, otherwise executes another block of code.
switch
Evaluates an expression and executes a block of code depending on a matching case.
?:
Ternary operator that provides a compact way to write an if...else statement.

Loops

for
Executes a block of code for a specified number of times.
while
Executes a block of code as long as a specified condition is true.
do...while
Similar to while loop, but executes the block of code once before checking the condition.
for...in and for...of
Iterates over iterable objects like arrays, strings, etc.

Arrays

push()
Adds one or more elements to the end of an array.
pop()
Removes the last element from an array and returns that element.
length
Returns the number of elements in an array.
