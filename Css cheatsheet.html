Selectors

* {}
Selects all elements on the page.
#id {}
Selects an element by its id attribute.
.class {}
Selects elements with a specific class attribute.
element {}
Selects all elements of a specific type.
element, element {}
Selects multiple elements.
parent > child {}
Selects a direct child of an element.

Properties

color
Sets the text color.
font-size
Sets the size of the font.
background-color
Sets the background color of an element.
border
Sets the border properties of an element.
margin
Sets the margin properties of an element.
padding
Sets the padding properties of an element.
display
Specifies the display behavior of an element.

Pseudo-classes

:hover
Applies styles when an element is hovered over.
:active
Applies styles when an element is being activated by the user.
:focus
Applies styles when an element is focused.
:nth-child()
Selects elements based on their position in a parent element.

Units

px
Pixels, a fixed-size unit.
%
Percentage, relative to the parent element.
em
Relative to the font-size of the element.
rem
Relative to the font-size of the root element.
vw
Viewport width, 1vw is equal to 1% of the viewport width.
vh
Viewport height, 1vh is equal to 1% of the viewport height.

Box Model

width
Sets the width of an element.
height
Sets the height of an element.
margin
Sets the margin properties of an element.
border
Sets the border properties of an element.
padding
Sets the padding properties of an element.

Flexbox

display: flex;
Specifies a flex container.
flex-direction
Defines the direction of the flex container.
justify-content
Aligns flex items along the main axis.
align-items
Aligns flex items along the cross axis.
flex-grow
Specifies how flex items should grow.
flex-shrink
Specifies how flex items should shrink.

Grid

display: grid;
Specifies a grid container.
grid-template-columns
Defines the columns of the grid.
grid-template-rows
Defines the rows of the grid.
grid-gap
Specifies the gap between grid items.
justify-items
Aligns grid items along the inline axis of the grid.
align-items
Aligns grid items along the block axis of the grid.
