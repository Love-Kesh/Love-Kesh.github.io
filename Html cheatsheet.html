Document Structure

<!DOCTYPE html>
This tells the browser that the document is written in HTML.
<html>
It wraps the entire HTML content of the page.
<head>
This contains meta-information about the document, such as the title and character encoding.
<title>
It sets the title of the webpage, which appears in the browser tab.
<body>
This contains the main content of the webpage that users see.

Text

<h1>
These are headings of different sizes, from largest (h1) to smallest (h6).
<p>
It creates a paragraph of text.
<span>
It is used to apply styles to small pieces of text.
<div>
It divides the content into sections or divisions.

Links

<a>
This creates a hyperlink, allowing users to navigate to another webpage when clicked.

Images

<img>
It embeds an image into the webpage.

Lists

<ul>
This creates an unordered list, typically represented by bullet points.
<ol>
It creates an ordered list, where each item is numbered.
<li>
It defines a list item within either an unordered or ordered list.

Tables

<table>
It creates a table to display data.
<tr>
This defines a table row.
<td>
It represents a cell within a table, containing data.
<th>
It defines a header cell within a table.

Forms

<form>
It creates a form for user input.
<input>
This creates various types of input fields, such as text fields or checkboxes.
<textarea>
It creates a multi-line text input field.
<button>
It creates a clickable button.
<select>
This creates a dropdown list.
<option>
It defines an option in a dropdown list.
